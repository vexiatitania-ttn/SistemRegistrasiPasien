# Sistem Registrasi Pasien Klinik

Program berbasis C++ untuk melakukan registrasi pasien pada sebuah klinik. Program dibuat menggunakan konsep dasar struktur data dan algoritma.

## Fitur

- Registrasi pasien
- Menampilkan seluruh data pasien
- Searching berdasarkan Nomor Rekam Medis atau Nama
- Sorting berdasarkan waktu registrasi
- Edit data pasien
- Hapus data pasien
- Validasi input

## Bahasa Pemrograman

- C++

## Konsep yang digunakan

- Struct
- Array
- Function
- Bubble Sort
- Sequential Search
- Input Validation
- String Manipulation

## Cara Menjalankan

Compile:

```bash
g++ main.cpp -o registrasi
